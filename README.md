## 📌 주요 기능
사이트 플로우 : 로그인 -> 포스팅된 내역 및 포스팅 쓰기 <-> 프로필  

**로그인**  
  - 소셜 회원가입 & 로그인(구글, 페이스북)
  - 이메일 회원가입 & 로그인
  - 첫 가입시 기본 닉네임 Anonymous, 프로필 Anonymous 사진 설정
  
**메인화면***
  - firebase 실시간 데이터 베이스로 새로고침 없이 등록되는 게시글 쓰기와 읽기
  - 본인이 등록한 게시글 수정, 삭제
  - 이미지 업로드 & 미리보기  
  - _**<span style="color:red">[update] 글작성폼에 emoji picker 추가</span>**_

**프로필**

  - 로그아웃
